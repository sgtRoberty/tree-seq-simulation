Scripts to simulate trees in R (FossilSim) and alignments in IQ-TREE \
1. Go to the `fossilWithSeqs` folder \
2. `Rscript Rscripts/simulateTreesSeqs.R`. Modify `Rscripts/simulateTreesSeqs.R` to define branching and clock parameters of your choice and IQ-TREE alignment simulation settings.\
2. `bash submit_all_reps.sh` \
